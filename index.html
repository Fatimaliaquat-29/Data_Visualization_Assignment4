Hierarchical Data & Social Network Connectivity

Software class hierarchies visualized as interactive trees. Karate club social relationships visualized as force-directed graphs.

Tree Visualizations

Source: Flare Dataset. The data describes the class hierarchy of the Flare ActionScript visualization library. The strict parent-child structure makes it ideal for tree-based layouts.

Layout 1 · Static Tree

Kamada-Kawai Layout

A force-directed algorithm that models the tree as a system of springs, aiming to preserve graph-theoretic distances in the geometric space.

Layout 2 · Static Tree

Spring Layout

A standard force-directed layout where connected nodes attract each other, allowing the hierarchical clusters to emerge organically.

Graph Visualizations

Source: Zachary's Karate Club. A classic social network dataset detailing friendships between 34 members of a karate club. This dataset is highly interconnected and non-hierarchical, making it perfect for graph network layouts.

Layout 1 · Static Graph

Circular Layout

Places all 34 members in a large ring. Useful for highlighting the density of connections across the entire club without any node occlusion.

View Visualization
Layout 2 · Static Graph

Spring Layout

Force-directed placement beautifully reveals the two distinct factions (Mr. Hi and the Officer) that the club ultimately split into over a conflict.
